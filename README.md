# Welcome to Address Book! 

Hello there! Thank you for checking out our little web application. Address Book is a cozy, user-friendly platform designed to help you manage your contacts and keep track of important information in a simple, elegant way.

## What is Address Book?

This is a static web app built with love, featuring a clean interface for organizing your address book entries. Whether you're jotting down notes or keeping tabs on your contacts, we've made it easy and enjoyable.

## Features You'll Love

- **Home Dashboard**: A welcoming landing page with helpful information cards
- **User Authentication**: Simple sign-up and login to keep your data personal
- **About Page**: Learn more about the app and its creators
- **Support Page**: Get help whenever you need it
- **Real-Time Clock**: Always know the current time and date
- **Responsive Design**: Looks great on any device

## Getting Started

We're excited for you to try it out! Here's how to get up and running:

1. Download or clone this repository to your local machine
2. Open the `index.html` file in your favorite web browser
3. Start exploring and enjoy!

## Built With Care Using

- HTML5 for structure
- CSS3 for styling
- Vanilla JavaScript for interactivity
